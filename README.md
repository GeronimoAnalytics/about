# Jeroen — Portfolio

Personal portfolio site for Jeroen, Senior Data Analyst and Power BI Developer. Built with Astro and Tailwind CSS as a professional reference to showcase my skills, experience, certifications, and projects in the data & analytics space.

## About

This portfolio highlights my expertise in:
- Data analysis and visualization
- Power BI development (Desktop, Service, DAX, Power Query)
- Data modeling and ETL pipeline management
- SQL, Databricks, Python, and DBT

It includes an `/agents` page providing context for AI agents to assess whether my skills match a visitor's needs.

## Built With

- **[Astro](https://astro.build/)** - Static site generator
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[Tabler Icons](https://tabler.io/icons)** - Icons
- **TypeScript** - Type-safe configuration

## Local Development

```bash
npm install
npm run dev
```

## Configuration

All content is managed through `src/config.ts` — personal info, skills, projects, experience, education, and social links.

## Deployment

Can be deployed to any static hosting service (Netlify, Vercel, GitHub Pages, Cloudflare Pages, etc.). See the [Astro deployment guides](https://docs.astro.build/en/guides/deploy/).
